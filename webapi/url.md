# url

## URL

```js
new URL(url, [base])
```

- 至少有一个参数。
- 只有一个参数时，参数必须是 absolute url
- 有两个参数时，第二个参数必须是 absolute url（href string 或 URL object）；若第一个参数是 absolute url 则忽略第二个参数。
